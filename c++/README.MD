## CRP编程语言手册
- 此版本基于c++运行，语法有所改变

by dzk

---

### 简介
- CRP语言是基于c++运行的类似汇编语言的编译型语言，力求简洁。

- CRP语言由python前端编译为二进制文件，与c++后端执行，执行效率高。

- CRP语言处理16位无符号数据。

### 命令
- CRP语言的命令非常精简
- 现在懒得写手册，命令参考上一版本，差不多，`compile.py`里的源代码应该看得到，再不行看看`main.cpp`

### 运行
- 在`compile.py`的`code`地方编写
- 运行后生产代码文件`example.bin`
- 编译`main.cpp`并运行
